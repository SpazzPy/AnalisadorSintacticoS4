# AnalisadorSintacticoS4

Este analizador sintáctico abrirá la terminal y en él podrás realizar cálculos aritméticos

### Que puede hacer:
1. Sumar ```1+1```
2. Restar ```1-1```
3. Multiplicación ```1*1```
4. División ```1/1```
5. Potenciación ```1^1```
6. Respetara el orden de los operadores
7. Números negativos ```-1```
